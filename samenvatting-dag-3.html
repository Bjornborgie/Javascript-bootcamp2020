Expressions as .length give acces to the property of the value almost all Javascript expressions has properties except 0 and undefined. there are 2 main ways to acces a property:

  1. with a dot like value.x
  2. and brackets likevalue[x]
but not necessarily the same property. the dot notation looks literally for the name after the dot, with square bracketsexpression between the brackets is evaluated. elements in an array are stored as the array properties, using numbers as property names you cant use the dot notation for numbers. Methods

Both string and array values contain, in addition to the length property, a number of properties that hold function values. toUppercase is a good example, when it is called it will convert a copy of the string in uppercases after is has send it to its function. Properties that contain functions are called methods.
